# Meet in the Middle

Find where two people can both live based on public-transport commute time to each of their jobs.

Drop two pins on the map — one per job — set how far each person is willing to travel, and the **green zone** shows every neighbourhood reachable by both. Works in any city with OpenStreetMap transit coverage.

**[→ Open the app](https://louis-josso.github.io/CityDistance/)**

---

Built with [Leaflet](https://leafletjs.org/), [Turf.js](https://turfjs.org/), and the [Geoapify Isoline API](https://www.geoapify.com/isoline-api/). No backend — everything runs in the browser.
